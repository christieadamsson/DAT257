# DAT257
## Agile Software Project Management


### Set up:
**Create and activate a viritual env in the backend folder**

**Install dependencies from requirements.txt**

**Create a .env file in your backend and add your Spoonacular API key formatted like this:**

SPOONACULAR_API_KEY=your_api_key_here

**Start the backend:**

uvicorn main:app --reload --port 8000

**Open a new terminal in the sustainablerecipefinder directory start the frontend**

npm run dev
