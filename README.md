# DAT257 - Agile Software Project Management

### Set up:
1 Create and activate a viritual env in the backend folder

2 Install dependencies from requirements.txt

3 Create a .env file in your backend and add your Spoonacular API key formatted like this:
```bash
SPOONACULAR_API_KEY=your_api_key_here
```
4 Start the backend:
```bash
uvicorn main:app --reload --port 8000
```
5 Open a new terminal in the sustainablerecipefinder directory start the frontend
```bash
npm run dev
```
